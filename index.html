Under the hood

Local-first by design.

  • 1
    Watches

    Detects new lines in your Claude Code JSONL files via filesystem events (debounced 2s).

  • 2
    Aggregates

    Computes tokens, cache rate, cost, model mix, burn rate — all in the extension host.

  • 3
    Renders

    Pushes state to a Preact webview. CSP-locked, no external resources beyond fonts.

  • 4
    Optional OAuth

    If enabled, fetches quota from api.anthropic.com using your existing Claude token.

# Install — one command. $ code --install-extension toksight.toksight # Or search "TokSight" in the Extensions panel. # Reload window. Pulse icon appears in your activity bar.
